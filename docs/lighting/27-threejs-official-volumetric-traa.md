## Overview

Part of the latest Three.js updates, these WebGPU examples demonstrate advanced volumetric lighting and god-rays. View source code in browser dev tools or GitHub.

## Key Techniques

- Temporal anti-aliasing (TRAA) for smooth raymarched volumes
- Custom shaders for light scattering
- Integration with post-processing for fog and beams
- WebGPU implementation

## Why Relevant

Ideal for moody atmospheres—volumetric effects create stylized fog, light shafts, and dramatic god-rays, perfect for ethereal or cinematic scenes. Official examples with production-quality code.
